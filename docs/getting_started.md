# Getting Started with Memory Maker

## Initial Settings

| Task | Location |
|------|----------|
| Update club info | Profile → About Club |
| Add an excomm team | Settings → Excomm |
| Add club members as users and assign auth-roles | Settings → Users |
| Add ticket types | Settings → Tickets |

---

## Meeting Management

### Before the Meeting

| Task | Location |
|------|----------|
| Create a meeting | Agenda → New |
| Update sessions | Agenda → Edit |
| Assign session owners (book meeting roles) | Agenda → Edit or Booking |
| Add project info for project sessions (prepared speeches) | Agenda → Edit |
| Generate meeting slides | Agenda |

### During the Meeting

| Task | Location |
|------|----------|
| Add/update roster entries | Roster |
| Add table topics speakers | Agenda → Edit |
| Start voting | Agenda |

### After the Meeting

| Task | Location |
|------|----------|
| Analyze the meeting reports | Voting / Roster |
| Add video media links | — |
| Complete levels | — |
